How to Use

          
Compact messy webpage copy into clean, paste-ready prompts — entirely on your device.

        


        

          

            
Quick start

            
    
              
  1. Select text on any webpage (at least your minimum character threshold).
    2. 
              
  2. Copy with Ctrl+C when auto-compact is on, or press Alt+Shift+C to compact without copying first.
    3. 
              
  3. Paste the cleaned Markdown into Claude, ChatGPT, Gemini, or any AI chat.
    4.

Other ways to compact

            
    
              
  • Context menu — Select text, right-click, choose SynapseClean: Compact for AI.
    • 
              
  • Extension popup — Toggle protection and auto-compact; open Prompt Console for full policy.
    • 
              
  • Compaction toast — After processing, a toast shows chars saved; click to copy again if needed.
    • 
            

          


          

            
Tips

            
    
              
  • Raise the minimum character threshold if small selections should copy unchanged.
    • 
              
  • Try Markdown, outline, or bullets output in Compaction Policy.
    • 
              
  • Open Refinement to pick a profile (AI Prompt, Product Spec, etc.) or edit Gemini prompts directly.
    • 
              
  • Enable Gemini Nano for structured refinement when Chrome 127+ and on-device AI are available.
    • 
              
  • Without Gemini, SynapseClean applies a default AI-ready Markdown template.
    •
